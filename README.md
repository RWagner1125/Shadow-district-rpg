# Shadow District RPG

A browser-based text RPG/MMO prototype where players begin as lowborn commoners and rise through work, reputation, education, guilds, town-building, and class progression.

## Current Features

- Character UI
- Visible stats
- Hidden soul reading concept
- Origin quest
- Recognized Citizen requirements
- Town project display
- Guild request display
- Basic JavaScript quest reward button

## Built With

- HTML
- CSS
- JavaScript
- GitHub Pages
